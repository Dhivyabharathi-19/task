CSS overflow-x and overflow-y Property Examples

1. overflow-x: visible; overflow-y: visible;

Content overflows both horizontally and vertically, but is visible outside the box.

2. overflow-x: hidden; overflow-y: scroll;

Horizontal overflow is hidden; vertical overflow can be scrolled.

3. overflow-x: scroll; overflow-y: hidden;

Horizontal overflow can be scrolled; vertical overflow is hidden.

4. overflow-x: auto; overflow-y: auto;

Scrollbars appear only when content overflows.
