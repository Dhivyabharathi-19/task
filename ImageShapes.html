CSS shape-outside Examples

Shape: circle()
This text wraps around a circular shape defined with `shape-outside: circle()`.
Shape: ellipse(50% 40% at 50% 50%)
The text flows around an elliptical shape, which is wider than it is tall.
Shape: polygon(50% 0%, 0% 100%, 100% 100%)
This example creates a triangle-like shape with three points.
Shape: inset(10px 20px 10px 20px)
Text wraps around a rectangle inset inward from each edge.
Shape: content-box
Uses the content area as the shape boundary. Padding does not affect the shape.
Shape: border-box
Includes border in the shape boundary.
Shape: margin-box
Uses the margin edge for text wrapping. Works best with defined margins.
